== Description ==

Make your website faster by saving a cached copy of it.

= Why this plugin? =

This plugin will reduce you site loading time by loading site from saving a cached copy of it.

= Who can use this plugin? =

Anyone who want to add caching on website.

Note
This plugin works only on WordPress sites.

Support
Contact our support

== Installation ==

**Automatic install:**

1. Log into your WP-Admin.
2. Navigate to 'Plugins' > 'Add New'.
3. Search for "proisp-vcache" and install it from there.

**Manual install:**

1. Upload the 'proisp-vcache.zip' and extract its contents to the '/wp-content/plugins/' directory or install via the WP plugins panel in your WordPress admin dashboard.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. That's it! You should now be able to use this.
